Question 1

Những vị trí nào có thể đặt Javascript trong file HTML và dặt theo cú pháp nào? Nên đặt Javascript tại vị trí nào để có hiệu suất tốt nhất? Cho ví dụ minh họa.

Javascript được đặt ở đâu?

Javascript có thể được đặt trong <body> hoặc <head> hoặc cả hai.

Javascript trong <body>

Ví dụ


<!DOCTYPE html>
<html>
<body> 
<h1>A Web Page</h1>
<p id="demo">A Paragraph</p>
<button type="button" onclick="myFunction()">Try it</button>
<script>
function myFunction() {
document.getElementById("demo").innerHTML = "Paragraph changed.";
}
</script>
</body>
</html>
Đặt tập lệnh ở dưới cùng của phần tử <body> sẽ cải thiện tốc độ hiển thị, vì việc giải thích tập lệnh làm chậm màn hình.

Javascript trong <head>

Ví dụ


<!DOCTYPE html>
<html>
<head>
<script>
function myFunction() {
  document.getElementById("demo").innerHTML = "Paragraph changed.";
}
</script>
</head>
<body>
<h1>A Web Page</h1>
<p id="demo">A Paragraph</p>
<button type="button" onclick="myFunction()">Try it</button>
</body>
</html>

External Javascript

Javascript có thể được định nghĩa từ một file .js độc lập

Ví dụ

Nội dung External File

function myFunction() {
document.getElementById("demo").innerHTML = "Paragraph changed.";
}
Định nghĩa file

<script src="myScript.js"></script>

Có thể định nghĩa file Javascript ở <head> hoặc <body>

Question 2

Trong Javascript có những kiểu dữ liệu nào? Làm thế nào để tạo một biến để lưu trữ giá trị tương ứng với những kiểu dữ liệu đó?.

Javascript có kiểu dữ liệu động

Ví dụ


var x;           // x có kiểu undefined
x = 5;           // Hiện tại x kiểu Number
x = "John";      // Hiện tại x kiểu String

Kiểu string

Biến string được đặt trong dấu ngoặc (đơn hoặc kép)

Ví dụ


var carName1 = "Volvo XC60";
var carName2 = 'Volvo XC60';

Kiểu Numbers

JavaScript chỉ có một kiểu dữ liệu số

Ví dụ


  var x1 = 34.00;
  var x2 = 34;

Kiểu booleans

Có hai giá trị true hoặc false

Ví dụ


var x = 5;
var y = 5;
var z = 6;
(x == y)  //Returns true
(x == z)  //Returns false

Kiểu arrays

Ví dụ


<!DOCTYPE html>
<html>
<body>

<h2>JavaScript Arrays</h2>

<p>Array indexes are zero-based, which means the first item is [0].</p>

<p id="demo"></p>

<script>
var cars = ["Saab","Volvo","BMW"];

document.getElementById("demo").innerHTML = cars[0];
</script>

</body>
</html>

Kiểu opject

Ví dụ


<!DOCTYPE html>
<html>
<body>

<h2>JavaScript Objects</h2>

<p id="demo"></p>

<script>
var person = {
  firstName : "John",
  lastName  : "Doe",
  age     : 50,
  eyeColor  : "blue"
};

document.getElementById("demo").innerHTML =
person.firstName + " is " + person.age + " years old.";
</script>

</body>
</html>

Kiểu undefined

Một biến chưa có giá trị thì có kiểu và giá trị là undefined

Ví dụ


var car;  //Giá trị của car và kiểu dữ liệu của car là undefined

Question 3

Làm thế nào để truy xuất (lấy giá trị) trong 1 thẻ HTML bất kỳ và gán vào 1 biến để trong Javascript theo ID, theo Class, theo Name, theo tên thẻ HTML và theo CSS Selector (ví dụ .class, #id)? Cho ví dụ minh họa.

DOM Element trong javascript

  • Tìm thẻ HTML theo ID
  • Tìm thẻ HTML theo class
  • Tìm thẻ HTML theo tên thẻ
  • Tìm thẻ HTML theo selector

Ví dụ:


<html>
<body>
  
  <input type="text" value="Freetuts.net" id="website" />
  
  <script language="javascript">
    // Lấy thẻ input
    var element = document.getElementById('website');
    
    // Lấy giá trị của thẻ input
    document.write(element.value);
  </script>
</body>
</html>

Tìm thẻ HTML theo class

Ví dụ:


<html>
  <body>
    
    <input type="text" value="Freetuts.net" class="website"/>
    
    <script language="javascript">
      // Lấy thẻ input
      var element = document.getElementsByClassName('website');
      
      // Lấy giá trị của thẻ input
      document.write(element[0].value);
    </script>
  </body>
</html>

Tìm thẻ HTML theo tên thẻ

Ví dụ:


<html>
  <body>
    
    <input type="text" value="Freetuts.net"/>
    
    <script language="javascript">
      // Lấy thẻ input
      var element = document.getElementsByTagName('input');
      
      // Lấy giá trị của thẻ input
      document.write(element[0].value);
    </script>
  </body>
</html>

Tìm thẻ HTML theo selector

Ví dụ:


<html>
  <body>
    <input type="text" value="thẻ không cần lấy" class="website"/>
    <div>
        <input type="text" value="Thẻ Cần Lấy" class="website"/>
        <input type="text" value="thẻ không cần lấy"/>
    </div>
    <script language="javascript">
      // Lấy thẻ input
      var element = document.querySelectorAll("div input.website");
      
      // Lấy giá trị của thẻ input
      document.write('<h3>Nội dung thẻ cần lấy là: ' + element[0].value + '</h3>');
    </script>
  </body>
</html>

Xác định các lệnh Javascript tương ứng để thực hiện các yêu cầu sau:

  • Viết ghi chú trong Javascript (1 dòng, nhiều dòng)
  • Hiển thị thời gian hiện tại (từ năm đến giây theo múi giờ GMT+7 - Việt Nam)
  • Viết 1 hàm (function) để tính tổng giá trị của 2 số ngẫu nhiên từ 1 đến 100
  • Đổi màu nền hay hình nền của trang

  The code below will change
  the heading with id = "myH"

  */

Hiển thị thời gian hiện tại (Việt Nam)

Source code


function startTime(){
  var asiaTime = new Date().toLocaleString("en-US", {timeZone: "Asia/Ho_Chi_Minh"});
  asiaTime = new Date(asiaTime);

  var year=asiaTime.getFullYear();
  var month=asiaTime.getMonth();
  var day=asiaTime.getDate();
  var hours=asiaTime.getHours();
  var minutes=asiaTime.getMinutes();
  var seconds=asiaTime.getSeconds();

  month++;
  var result=year+" - " + month + " - " + day + "   " + hours + ":" + minutes + ":" + seconds + "   GMT+7";
  
  document.getElementById("question4_2").innerHTML = result;
  var t = setTimeout(startTime, 500);
}
