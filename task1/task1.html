1. Có những cách đặt Javascript nào trong file HTML? Cho ví dụ minh họa

Javascript được đặt ở đâu?

Javascript có thể được đặt trong <body> hoặc <head> hoặc cả hai.

Javascript trong <body>

Ví dụ


                <!DOCTYPE html>
                <html>
                <body> 
                <h1>A Web Page</h1>
                <p id="demo">A Paragraph</p>
                <button type="button" onclick="myFunction()">Try it</button>
                <script>
                function myFunction() {
                document.getElementById("demo").innerHTML = "Paragraph changed.";
                }
                </script>
                </body>
                </html>
Đặt tập lệnh ở dưới cùng của phần tử <body> sẽ cải thiện tốc độ hiển thị, vì việc giải thích tập lệnh làm chậm màn hình.

Javascript trong <head>

Ví dụ


                  <!DOCTYPE html>
                  <html>
                  <head>
                  <script>
                  function myFunction() {
                    document.getElementById("demo").innerHTML = "Paragraph changed.";
                  }
                  </script>
                  </head>
                  <body>
                  <h1>A Web Page</h1>
                  <p id="demo">A Paragraph</p>
                  <button type="button" onclick="myFunction()">Try it</button>
                  </body>
                  </html>

External Javascript

Javascript có thể được định nghĩa từ một file .js độc lập

Ví dụ

Nội dung External File

                function myFunction() {
                document.getElementById("demo").innerHTML = "Paragraph changed.";
                }
Định nghĩa file

              <script src="myScript.js"></script>

Có thể định nghĩa file Javascript ở <head> hoặc <body>

2. Có thể hiển thị dữ liệu từ Javascript với những cách nào? Cho ví dụ minh họa.

Hiển thị dữ liệu từ Javascrjpt

Có nhiều cách khác nhau để hiển thị dữ liệu:

  • Dùng innerHTML
  • Dùng document.write()
  • Dùng window.alert()
  • Dùng console.log()

Dùng innerHTML

Khi sử dụng document.getElementById().innerHTML, nội dung mà bạn muốn hiển thị sẽ được hiển thị bên trong một phần tử xác định.

Ví dụ:

              
                <!DOCTYPE html>
                <html>
                <body>

                <h1>My First Web Page</h1>
                <p>My First Paragraph</p>

                <p id="demo"></p>

                <script>
                document.getElementById("demo").innerHTML = 5 + 6;
                </script>

                </body>
                </html>

Dùng document.write()

Khi sử dụng document.write(), nội dung mà bạn muốn hiển thị sẽ được hiển thị ngay đúng vị trí mà nó được đặt trong trang web.

Ví dụ:

              
                <!DOCTYPE html>
                <html>
                <body>

                <h1>My First Web Page</h1>
                <p>My first paragraph.</p>

                <script>
                document.write(5 + 6);
                </script>

                </body>
                </html>

Dùng window.alert()

Khi sử dụng alert(), nội dung mà bạn muốn hiển thị sẽ được hiển thị bên trong một hộp thoại thông báo

Ví dụ:

              
                <!DOCTYPE html>
                <html>
                <body>

                <h1>My First Web Page</h1>
                <p>My first paragraph.</p>

                <script>
                window.alert(5 + 6);
                </script>

                </body>
                </html>

Dùng console.log()

Nhằm mục đích debug

Ví dụ:

              
                <!DOCTYPE html>
                <html>
                <body>

                <script>
                console.log(5 + 6);
                </script>

                </body>
                </html>

3. Làm thế nào để truy xuất (lấy giá trị) 1 thẻ HTML bất kỳ theo ID, theo Class, theo Name, theo tên thẻ HTML và theo CSS Selector? Cho ví dụ minh họa

DOM Element trong javascript

  • Tìm thẻ HTML theo ID
  • Tìm thẻ HTML theo class
  • Tìm thẻ HTML theo tên thẻ
  • Tìm thẻ HTML theo selector

Ví dụ:

              
                <html>
                <body>
                  
                  <input type="text" value="Freetuts.net" id="website" />
                  
                  <script language="javascript">
                    // Lấy thẻ input
                    var element = document.getElementById('website');
                    
                    // Lấy giá trị của thẻ input
                    document.write(element.value);
                  </script>
                </body>
              </html>

Tìm thẻ HTML theo class

Ví dụ:

              
              <html>
                <body>
                  
                  <input type="text" value="Freetuts.net" class="website"/>
                  
                  <script language="javascript">
                    // Lấy thẻ input
                    var element = document.getElementsByClassName('website');
                    
                    // Lấy giá trị của thẻ input
                    document.write(element[0].value);
                  </script>
                </body>
              </html>

Tìm thẻ HTML theo tên thẻ

Ví dụ:

              
              <html>
                <body>
                  
                  <input type="text" value="Freetuts.net"/>
                  
                  <script language="javascript">
                    // Lấy thẻ input
                    var element = document.getElementsByTagName('input');
                    
                    // Lấy giá trị của thẻ input
                    document.write(element[0].value);
                  </script>
                </body>
              </html>

Tìm thẻ HTML theo selector

Ví dụ:

              
              <html>
                <body>
                  <input type="text" value="thẻ không cần lấy" class="website"/>
                  <div>
                      <input type="text" value="Thẻ Cần Lấy" class="website"/>
                      <input type="text" value="thẻ không cần lấy"/>
                  </div>
                  <script language="javascript">
                    // Lấy thẻ input
                    var element = document.querySelectorAll("div input.website");
                    
                    // Lấy giá trị của thẻ input
                    document.write('<h3>Nội dung thẻ cần lấy là: ' + element[0].value + '</h3>');
                  </script>
                </body>
              </html>
4.Xác định các lệnh Javascript tương ứng để thực hiện các yêu cầu sau:
  • Hiển thị ngày tháng và thời gian hiện tại.
  • Thay đổi nội dung trong 1 thẻ HTML đang có>
  • Hiển thị một số ngẫu nhiên (random) từ 1 đến 1000.
  • Cộng, trừ, nhân, chia, chia lấy phần dư.

Hiển thị ngày tháng và thời gian hiện tại

  • getFullYear(): Lấy năm
  • getMonth(): Lấy tháng
  • getDate(): Lấy ngày
  • getHours(): Lấy giờ
  • getMinutes(): Lấy phút
  • getSeconds: Lấy giây
  • getMilliseconds(): Lấy millisecond
  • getTime(): Lấy thời gian millisecod từ 1-1-1970
  • getDay(): Lấy ngày trong tuần (0-6)

Thay đổi nội dung trong 1 thẻ HTML đang có

Hiển thị một số ngẫu nhiên từ 1 đến 1000

Math.floor(Math.random() * 999) + 1

Cộng, trừ, nhân, chia, chia lấy phần dư

Các phép toán lần lượt: + - * / %
